# Cocktail AI App

## Overview

Cocktail AI App is a cloud-native application designed to help users discover, explore and manage cocktail recipes through an intuitive interface, while serving as a practical learning platform for modern cloud engineering and artificial intelligence technologies.

The project has two primary objectives:

1. Deliver a high-quality cocktail application with powerful search, discovery and recommendation capabilities.
2. Provide a long-term engineering project that demonstrates modern software architecture, AWS cloud services, AI integration, software engineering practices and continuous technical growth.

---

# Vision

Create a scalable, cloud-native cocktail platform that combines structured recipe data with AI-powered features to deliver personalised cocktail discovery, recommendations and insights.

The application will evolve incrementally from a locally developed Python application into a production-ready AWS-hosted solution incorporating serverless architecture, cloud-native services and generative AI capabilities.

---

# Project Goals

## Functional Goals

- Browse and search cocktail recipes.
- Search by ingredients.
- View detailed cocktail information.
- Support favourites and user preferences.
- Provide AI-assisted cocktail recommendations.
- Provide insights and analytics over the cocktail dataset.
- Support future administrative tooling and data management.

## Engineering Goals

- Learn and apply modern AWS services.
- Develop practical AI engineering skills.
- Build a portfolio-quality software project.
- Maintain high-quality documentation alongside the codebase.
- Demonstrate professional software engineering practices.

# Current Architecture

Current Status

🟡 Early Development

The project currently consists of a Python application with a modular backend architecture and supporting documentation.

Architecture documentation has been established early in the project to ensure design decisions, technical direction and system evolution remain well documented throughout development.

---

# Target Architecture

The long-term target architecture is expected to include services such as:

- React Frontend
- Amazon CloudFront
- Amazon API Gateway
- AWS Lambda
- Amazon DynamoDB
- Amazon Cognito
- Amazon Bedrock (future AI capabilities)
- Amazon CloudWatch

The architecture will evolve iteratively as the application matures.
